# GymPortal

GymPortal is an ASP.NET Core MVC application for Gyn website with memberships, classes, and bookings and account page.

## Features
- User registration and login
- Gym class management
- Membership system
- Booking system
- Customer profile management

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- Identity Authentication
- Bootstrap
- SQLite

## Getting Started
1. Clone the repo
2. Run database migrations
3. Start the application

## Start
1. Cd gymportal.web
2. Dotnet build
3. Dotnet run
